# 开始使用 Pump-Up

欢迎使用 Pump-Up！这份指南将帮助你快速上手。

## 下载安装

1. 打开 App Store
2. 搜索 "Pump-Up"
3. 点击下载安装

## 首次设置

首次打开 App 时，你需要：

- **创建账户**：使用邮箱注册或 Apple ID 登录
- **设置个人信息**：填写身高、体重等基本信息
- **授权 HealthKit**：允许 App 读取你的健康数据（可选）

> 所有个人数据都存储在你的设备本地，我们不会上传至任何服务器。

## 主要功能

### 锻炼追踪

记录你的每一次训练，支持：

- 力量训练
- 有氧运动
- HIIT 高强度间歇
- 瑜伽与柔韧性

### 冥想放松

多种冥想课程帮助你：

- 改善睡眠
- 减轻压力
- 提升专注力

### 健康数据

与 Apple HealthKit 深度集成：

- 步数统计
- 心率监测
- 睡眠分析
- 卡路里消耗
